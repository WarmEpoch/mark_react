# 沐享水印

沐享水印是一个使用Web技术实现的轻量级水印添加工具。本项目利用`canvas`和`html2canvas`技术，可以在网页上方便地添加水印。

## 特性

- 使用现代Web技术构建
- 基于Canvas和html2canvas实现
- 简单易用的API接口

## 环境要求

- Node.js
- npm

## 安装

确保您的环境已经安装了Node.js和npm。然后执行以下命令来安装项目依赖：

```bash
npm i
```

## 使用方法

启动开发服务器：

```bash
npm run dev
```

这将启动一个本地服务器，您可以在浏览器中访问和测试水印功能。
